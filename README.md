# Contatos – Modelo

## Sobre
Projeto desenvolvido para compreender e praticar o uso de métodos HTTP no front-end, orientado pelo professor Fernando Leonid.

## Descrição
Aplicação voltada para o gerenciamento de contatos, permitindo:
- Adicionar  
- Editar  
- Atualizar  
- Remover  

Cada contato pode conter nome, telefone e foto, com comunicação por requisições HTTP.
